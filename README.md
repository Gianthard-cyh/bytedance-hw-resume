# 字节跳动前端训练营作业1-个人简历
做了一个整体上模仿传统纸张媒体的个人简历，加入了一些比较复古的元素，例如标题下面的双横线和衬线体。

初稿是由Gemini3给出的，AI目前在前端领域的能力确实相当亮眼，能够快速生成符合要求的代码，同时样式挺好看的。在此基础上做了一些调整，例如颜色间距和一些逻辑。

相比于把所有的元素都写死成html，我把一些动态的内容（例如姓名、标题、联系方式等）都从json文件中读取，这样如果需要修改这些内容，只需要修改json文件即可，或许在之后可以做成一个简历生成应用。

做了打印适配，点击右下角打印按钮可以对页面进行打印。
