# 字节跳动前端训练营作业1 - 个人简历（HTML/CSS/JS）
这是一个以语义化 HTML 为基础、以少量可读性良好的 CSS 与原生 JavaScript 完成交互的小型静态简历页面，强调清晰的结构与可维护性，同时在不引入任何打包或框架的前提下提供折叠与打印等常用功能。

## 功能
- 每个章节的标题左侧提供一个三角形的折叠控制按钮，用户可以在不改变页面整体布局的情况下自由折叠或展开章节内容，以便在浏览时快速聚焦于感兴趣的部分。
- 页面在打印场景下自动应用专用样式，通过隐藏交互控件与装饰性阴影、释放页面宽度等方式，使打印输出保持清晰、规整并且易于阅读和保存。
- 网格布局针对不同尺寸的设备进行了简洁的响应式处理，在中等及以上屏幕尺寸下展示为主栏与侧栏的双列结构，而在较小的设备上则退化为单列结构以确保舒适的阅读体验。

## 实现细节
- 页面采用简洁的 CSS Grid 布局，并通过 `@media (min-width: 768px)` 这一断点在不同设备上切换列数，从而在桌面端呈现主栏与侧栏的双列视图，在移动端收敛为单列视图以保证阅读流畅。
- 通过 `.stack > * + * { margin-top: 1rem; }` 这一规则将相邻元素的间距统一为一致的垂直节奏，使得模块之间的层次更易分辨，同时避免了首元素产生不必要的顶部空白。
- 页面结构严格遵循语义化设计，核心区域使用 `header`、`main`、`section`、`aside`、`article` 与 `footer`，并在联系方式、时间与统计信息处分别使用 `address`、`time` 与 `dl/dt/dd`，以提升可访问性与机器可读性。
- 折叠控制使用原生 `button` 元素并以 `aria-expanded` 标识当前状态，脚本仅用于增强交互而不影响内容的可见性；在脚本不可用或未加载的情况下，页面仍以展开状态正常展示，确保渐进增强。
- 针对打印场景，`@media print` 会主动隐藏交互按钮与装饰性阴影并释放页面宽度，使输出更贴近纸面阅读的需求，从而在导出 PDF 或直接打印时维持统一的版式。
- 资源管理方面保持纯静态与零打包的策略，避免引入不必要的构建与依赖，使页面加载更轻更快，同时将样式数量控制在合理范围以便后续维护与复用。

## 使用
- 将项目下载或克隆到本地后，直接用浏览器打开 `index.html` 即可完成预览，无需安装任何依赖或运行开发服务器。
- 若需要调整样式或版式，只需在 `styles.css` 中修改相应的类名与规则即可；所有类名均以语义为导向，便于检索与理解。

## 部署
